# FOODco. — Food Ordering App

A modern dark-themed food ordering web app built with React, Vite, and Tailwind CSS.

---

## Project Structure

```
foodco/
├── index.html
├── package.json
├── vite.config.js
├── tailwind.config.js
├── postcss.config.js
├── src/
│   ├── main.jsx
│   ├── App.jsx
│   ├── index.css
│   ├── Context.jsx
│   ├── Context2.jsx
│   └── components/
│       ├── NavBar.jsx
│       ├── Hero/Hero.jsx
│       ├── Deal/Deal.jsx
│       ├── Menu/Menu.jsx
│       ├── Order/Order.jsx
│       ├── Team/Team.jsx
│       └── Footer/Footer.jsx
└── server/
    ├── app.js
    └── package.json
```

---

## Setup — Frontend

```bash
# From the root foodco/ folder:
npm install
npm run dev
```

The app will be available at **http://localhost:5173**

---

## Setup — Backend (Email Server)

```bash
cd server
npm install
npm run dev
```

The server runs at **http://localhost:4000**
